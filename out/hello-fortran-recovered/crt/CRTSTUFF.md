# CRT scaffolding (documentation only)

These functions were emitted into the binary by the toolchain
(libgcc / crtbegin.o / crtend.o). They are recovered here for
reference; the build does **not** include them — the real
runtime objects are linked by the C++ driver.

## `__do_global_dtors_aux`  @ `0x11a0`

- Demangled: `__do_global_dtors_aux`
- Role: `ctor_dtor`
- Confidence: 0.92

@brief Compiler-generated finalizer that runs the program's global destructors exactly once.
This is part of the standard CRT (C runtime) initialization/finalization
machinery emitted by the toolchain and registered into the .fini_array
section. It is invoked automatically at program/shared-object teardown
and should not be called directly by user code.
The function:
 - Returns immediately if the static guard @c completed.0 is already set,
   ensuring destructors run at most once.
 - If @c __cxa_finalize is available (non-NULL), calls it with
   @c __dso_handle so that any C++/destructor callbacks registered for
   this DSO via @c __cxa_atexit are executed.
 - Calls @c deregister_tm_clones() to undo the transactional-memory
   clone table registration performed by the matching constructor.
 - Sets the @c completed.0 guard so subsequent invocations are no-ops.
@return None.
@note Generated by the compiler/linker for every executable or shared
      library; paired with @c frame_dummy / @c register_tm_clones on the
      constructor side. Not intended to be called from application code.

```
static void __do_global_dtors_aux(void)
{
    if (completed.0)
        return;

    if (__cxa_finalize != NULL)
        __cxa_finalize(__dso_handle);

    deregister_tm_clones();
    completed.0 = 1;
}

```

## `frame_dummy`  @ `0x11e0`

- Demangled: `frame_dummy`
- Role: `entry_stub`
- Confidence: 0.90

@brief CRT initialization stub emitted by GCC.
Standard `frame_dummy` routine produced by the GCC toolchain and linked
into the program's `.init_array` (or legacy `.ctors`) section. It is
invoked automatically by the C runtime startup code before `main()` to
participate in per-translation-unit constructor/exception-frame setup.
In modern GCC output the body is effectively empty: any real work
(registering Java class records via `__JCR_LIST__`, or registering
transactional-memory clones) has been folded into companion helpers
such as `register_tm_clones`. This function is retained purely as a
placeholder so the constructor table layout remains compatible with
older runtimes.
@note Not intended to be called from user code; it is dispatched by the
      CRT (`__libc_csu_init` / `_init`) during process startup.
@note Takes no parameters and returns no value.

```
/* frame_dummy — standard CRT entry stub emitted by GCC.
 * Calls register_tm_clones()-style helper via __JCR_LIST__/deregistration logic.
 */
void frame_dummy(void)
{
}

```

## `_start`  @ `0x1100`

- Demangled: `_start`
- Role: `entry_stub`
- Confidence: 0.70

@brief ELF program entry point invoked by the kernel/dynamic loader.
This is the standard System V AMD64 `_start` stub. The compiler-generated
prologue (not expressible in portable C) extracts `argc`, `argv`, and the
stack-end pointer from the initial stack frame laid out by the kernel,
realigns `rsp` to a 16-byte boundary, zeros the frame-pointer chain, and
then tail-calls `__libc_start_main`, which performs C runtime
initialization, calls `main`, and ultimately invokes `exit` — it never
returns to `_start`.
The `init`, `fini`, and `rtld_fini` hook pointers are passed as NULL in
this build, so libc relies on its own constructor/destructor handling.
@return This function does not return. Control is transferred to
        `__libc_start_main`, and the trailing `__builtin_trap()` (compiled
        as `hlt`) is reached only if `__libc_start_main` erroneously
        returns, in which case the process aborts immediately.
@note Inputs are taken from the architectural registers populated by the
      assembly prologue: `rdi` (argc), `rsi` (argv), and `r9` (stack_end).
@note This entry point is referenced by the ELF header's `e_entry` field
      and is the first user-mode instruction executed after the dynamic
      linker (`/lib64/ld-linux-x86-64.so.2`) finishes relocation.

```
/*
 * ELF entry point. The compiler-generated _start is almost entirely an ABI
 * shim: it pulls argc/argv off the initial stack, aligns rsp to 16, zeros
 * the frame-pointer chain, and tail-calls __libc_start_main, which never
 * returns. The trailing `hlt` is a guard against __libc_start_main ever
 * returning by mistake.
 *
 * None of the stack/register marshalling is expressible in standard C, so
 * the body below only models the C-visible call.
 */

extern int main(int argc, char **argv, char **envp);
extern void __libc_start_main(int (*main)(int, char **, char **),
                              int argc, char **argv,
                              void (*init)(void), void (*fini)(void),
                              void (*rtld_fini)(void), void *stack_end);

void _start(void)
{
    /* argc, argv come from the initial stack laid out by the kernel; the
     * asm prologue extracts them before this call. */
    register int     argc      __asm__("rdi");
    register char  **argv      __asm__("rsi");
    register void   *stack_end __asm__("r9");

    __libc_start_main(main, argc, argv,
                      /* init       */ 0,
                      /* fini       */ 0,
                      /* rtld_fini  */ 0,
                      stack_end);

    __builtin_trap();   /* `hlt` — __libc_start_main is noreturn */
}

```

## `deregister_tm_clones`  @ `0x1130`

- Demangled: `deregister_tm_clones`
- Role: `ctor_dtor`
- Confidence: 0.55

@brief Compiler-generated crtstuff stub that deregisters Intel TM
       (Transactional Memory) clone tables at shared-object/program
       teardown.
This is the canonical `deregister_tm_clones` routine emitted by GCC's
crtstuff into every executable and DSO. It is wired into the
`.fini_array` (or invoked from `__do_global_dtors_aux`) and runs once
during shutdown. The function:
  1. Tests the file-scope guard `completed.0` against the address of
     `__TMC_END__`; if they are equal the routine has already executed
     (or no clone table was registered) and it returns immediately.
  2. Loads the GOT slot `_ITM_deregisterTMCloneTable_ptr`. Because the
     symbol is declared weak, this slot is NULL when libitm is not
     linked in, in which case the function returns without doing
     anything.
  3. Otherwise calls `_ITM_deregisterTMCloneTable(&__TMC_END__)` to
     tear down the TM clone table for this module.
@return None. The function has no observable side effects beyond the
        (optional) call into libitm's deregistration routine.
@note This function takes no parameters and is never called directly
      by user code; it is referenced only by the compiler-emitted
      shutdown machinery and should not be modified or invoked
      manually.
@note `completed.0` and `__TMC_END__` are local linker-defined
      symbols; their addresses (not their contents) are what is
      compared.

```
static void deregister_tm_clones(void)
{
    /* Canonical crtstuff stub emitted by the compiler.
     * Skip if we've already run, then call _ITM_deregisterTMCloneTable
     * via its GOT slot iff the symbol was actually resolved. */
    if (completed.0 == __TMC_END__)
        return;

    void (*deregister)(void *) = _ITM_deregisterTMCloneTable_ptr;
    if (deregister == NULL)
        return;

    deregister(completed.0);
}
```

## `register_tm_clones`  @ `0x1160`

- Demangled: `register_tm_clones`
- Role: `ctor_dtor`
- Confidence: 0.95

@brief Compiler-generated stub that registers the transactional-memory clone table.
Invoked automatically during program startup (typically from the `.init_array`
section, alongside `frame_dummy`). The function checks whether the linker
produced a non-empty TM clone table by comparing the bracketing symbols
`__TMC_LIST__` and `__TMC_END__`. If the table is empty, or if the weak
symbol `_ITM_registerTMCloneTable` was not provided by an ITM/libitm runtime,
the function returns without doing anything. Otherwise, it forwards the
table base and its element count (size in bytes divided by `sizeof(void *)`)
to `_ITM_registerTMCloneTable`.
This routine takes no parameters and has no return value; all inputs are
resolved through link-time symbols.
@return None.
@note Companion to `deregister_tm_clones`, which is called at shutdown.
@note `_ITM_registerTMCloneTable` is a weak reference, so a NULL check is
      required before invocation.

```
static void register_tm_clones(void)
{
    /* Compiler-generated stub: if __TMC_END__ - __TMC_LIST__ is non-zero
       and _ITM_registerTMCloneTable is available, register the clone table. */
    if (__TMC_END__ - __TMC_LIST__ == 0)
        return;

    if (_ITM_registerTMCloneTable == NULL)
        return;

    _ITM_registerTMCloneTable(__TMC_LIST__, (__TMC_END__ - __TMC_LIST__) / sizeof(void *));
}
```
